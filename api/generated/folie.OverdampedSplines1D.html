folie.OverdampedSplines1D¶

folie.OverdampedSplines1D(domain)[source]¶

Generate defaut model for estimation of overdamped Langevin dynamics.

Parameters:
knots: int of array

Either the number of knots to use in the spline or a list of knots. The more knots the more precise the model but it will be more expensive to run and require more data for precise estimation.
